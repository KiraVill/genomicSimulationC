◆ alleles

char* AlleleMatrix::alleles[CONTIG_WIDTH]

A matrix of SNP markers by lines/genotypes containing pairs of alleles eg TT, TA.

Use alleles[line index][marker index * 2] to get the first allele and alleles[lines index][marker index * 2 + 1] to get the second. If CONTIG_WIDTH lines are saved here, another AlleleMatrix is added to the linked list when there's a need to save more.
