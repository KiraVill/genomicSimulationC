◆ delete_bidirectional_iter()

void delete_bidirectional_iter ( BidirectionalIterator *  it)

Deletes a BidirectionalIterator object.

A BidirectionalIterator has no heap memory to free, so calling this function is mostly unnecessary. The function will set all values in the struct to uninitialised/null values, and will set the iterator to think it is both at the start and end of its sequence, so that any next_* functions will not attempt to search for group member locations.

Parameters
itpointer to the struct whose data is to be cleared.

void delete_group ( SimData *  d,
const GroupNum  group_id 
)

Deletes all genotypes belonging to a particular group.

This includes all of their details. Persistent ids (used to track pedigree) will not be re-used.

Uses a call to condense_allele_matrix() to ensure that the SimData remains valid after deletion.

Parameters
dthe SimData struct on which to perform the operation

Deletes a RandomAccessIterator object and frees its memory.
