Combine a set of groups into one group.

The function does so by setting the group membership of every genotype belonging to one of the groups to the same group number.

Parameters
dthe SimData struct on which to perform the operation
list_lenthe number of groups to be combined
group_idsan array of group numbers containing the groups that are to be combined.
Returns
the group number of the new combined group.

◆ split_into_families()

void split_into_families ( SimData *  d,
int  group_id 
)

Split a group into a set of smaller groups, each of which contains the genotypes in the original group that share a particular pair of parents.

The number of new groups produced depends on the number of parent-combinations in the set of genotypes in the provided group.

Individuals with both parents unknown will be grouped together.

Note: this does not return the group numbers of all the newly created groups-of-one.

Parameters
dthe SimData struct on which to perform the operation
group_idthe group number of the group to be split
